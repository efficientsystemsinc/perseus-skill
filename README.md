# perseus skill

An [agent skill](https://www.skills.sh/) for **plain-English code search** with
[perseus](https://perseus.computer) — describe the code you're looking for and get ranked
`path:line` results with snippets in a few seconds, instead of grepping or clicking through files.

## Install

```bash
npx skills add HACKE-RC/perseus-skill
# or
gh skill install HACKE-RC/perseus-skill perseus
```

## What's in it

- `skills/perseus/SKILL.md` — when to reach for perseus over grep, how to phrase queries for a high
  hit rate, how to read the JSON results, and how to keep the index fresh.
